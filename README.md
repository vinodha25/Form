# Form Validation Web App

A simple web-based login form that validates user input in real-time using HTML and JavaScript.  
It checks if the entered username and password match the predefined credentials and gives feedback to the user.

## Features
- Username and password validation
- Alerts user for successful verification
- Alerts user for invalid login attempts
- Simple and beginner-friendly design

## Technologies Used
- HTML
- JavaScript
- CSS (optional, can style later)
